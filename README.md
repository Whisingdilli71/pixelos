Pixel OS is a retro‑styled, browser‑based operating system simulation.  
It recreates the look and feel of classic desktop environments with pixel‑art UI, draggable windows, a taskbar, and a Start Menu.

---

## Features

- **Start Menu** with flyout submenus (Programs, Accessories, Settings, Logout, Shut Down).
- **Taskbar** with Start button, open app buttons, and a live clock.
- **Apps**:
  - Notepad
  - Paint
  - Terminal
  - Browser
  - Clock
  - Settings
- **Login Page** (`index.html`):
  - Simple blue wallpaper background.
  - Editable username field.
  - Password field pre‑filled with a fixed random string (readonly).
  - Redirects to `os.html` (main OS) on successful login.

## Getting Started

1. Clone or download the repository.
2. Open `index.html` in your browser to start.
3. Enter any username. The password field is already filled and cannot be changed.
4. Click **Login** to enter the OS
